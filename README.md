# SpotifyScripts

## Running

To setup the repo, do the following:

1. Pull the repo
    1. `git clone https://github.com/RyanMKrol/SpotifyScripts`
1. Head to the repo
    1. `cd SpotifyScripts`
1. Install dependencies
    1. `npm install`
1. Pull credentials
    1. `npx dotenv-vault@latest pull`
1. Run a script
    1. `npm run script:generatePlaylist`
